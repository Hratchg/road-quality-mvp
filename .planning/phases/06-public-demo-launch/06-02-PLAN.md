# Plan 06-02: Pre-label assist via public pretrained pothole model

## Goal

Reduce CVAT hand-labeling burden 3–5× by generating YOLO-format
auto-suggested labels for all 158 LA images via the public pretrained
`keremberke/yolov8s-pothole-segmentation` model (which is the same base
model the project will fine-tune from per Phase 2 D-11). The operator
then opens these auto-labels in CVAT and CORRECTS them rather than
creating from scratch.

## Acceptance criteria

1. New script `scripts/prelabel.py` exists, runnable via
   `/tmp/rq-venv/bin/python scripts/prelabel.py`.
2. After running, every image under `data/eval_la/images/{train,val,test}/`
   has a corresponding `.txt` under `data/eval_la/labels/<split>/<image_id>.txt`
   containing valid YOLO format (`<class_id> <x_center> <y_center> <width> <height>`,
   all normalized 0–1, space-separated, class_id=0 for "pothole").
3. Labels with zero detections are written as empty files (not missing).
4. Commit message references the model repo + image count + total bbox count.
5. Out of scope: any change to `data_pipeline/detector_factory.py`, the
   model itself, or `data/eval_la/data.yaml`.

## Tasks

1. Write `scripts/prelabel.py`:
   - Load `keremberke/yolov8s-pothole-segmentation` via `huggingface_hub.hf_hub_download` then `ultralytics.YOLO()`
   - Iterate `data/eval_la/images/{train,val,test}/*.jpg`
   - For each image: model.predict(image, conf=0.25), get normalized xywh from results.boxes.xywhn
   - Write `<class_id=0> <x_c> <y_c> <w> <h>` lines to `data/eval_la/labels/<split>/<image_id>.txt`
   - Empty file if no detections

2. Run the script. Expected: ~5–10 min for 158 images on M-series CPU.

3. Spot-check: pick 3 random images, view the .txt files, verify boxes
   are sensible (e.g., a bounding box that's the entire image is a red flag).

4. Commit `data/eval_la/labels/` (the .txt files) plus `scripts/prelabel.py`
   atomically per GSD convention.

## Verification

```bash
# Every image has a label file
diff <(find data/eval_la/images -name '*.jpg' | sed 's|images/|labels/|; s|\.jpg$|.txt|' | sort) \
     <(find data/eval_la/labels -name '*.txt' | sort)

# Total bbox count
find data/eval_la/labels -name '*.txt' -exec cat {} + | wc -l

# Per-split counts
for split in train val test; do
  total=$(find data/eval_la/labels/$split -name '*.txt' | wc -l)
  with_boxes=$(find data/eval_la/labels/$split -name '*.txt' -size +0c | wc -l)
  echo "$split: $total files, $with_boxes with detections"
done
```
